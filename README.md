## Descripción del Proyecto

En este repositorio adjuntamos las multiples calculadoras de actividades que hicimos a lo largo del cuatrimestre

## Identidad Visual 

El proyecto se distingue por una estética única denominada **"Dark Academia / RPG Fantasy"**. A diferencia de las calculadoras tradicionales, esta interfaz busca la inmersión del usuario mediante:

* **Materialidad:** Texturas que simulan piedra oscura, pizarra y pergamino antiguo.
* **Paleta de Colores:** Tonos profundos (#101010, #2c2c2c) contrastados con acentos en oro viejo (#d0b07a) y bronce (#cfa85d).
* **Tipografía:** Uso de fuentes clásicas y rúnicas (*Cinzel Decorative*, *EB Garamond*) para títulos y textos.
* **Interactividad:** Efectos de brillo, sombras y transiciones suaves que simulan una interfaz mágica o antigua.

## Herramientas Incluidas

El portafolio consta de las siguientes calculadoras y visualizadores:

### 1. Visualizador de la Recta Numérica
Herramienta gráfica para ubicar y clasificar números reales.
* **Funcionalidades:** Clasificación automática (Naturales, Enteros, Racionales, Irracionales), visualización de "gemas" en la recta y cálculo de distancia entre puntos.

### 2. Conversor de Números Complejos
Calculadora omnidireccional para transformar números complejos.
* **Formatos soportados:** Cartesiana ($a+bi$), Polar ($r(\cos \theta + i \sin \theta)$) y Exponencial ($re^{i\theta}$).
* **Características:** Detecta entradas matemáticas como `sqrt(2)` o `pi`.

### 3. Calculadora de Productos Notables
Desglose paso a paso de las reglas algebraicas de expansión.
* **Casos:** Binomio al cuadrado, Binomio al cubo, Conjugados y Término común.
* **Extra:** Incluye visualización geométrica (cálculo de áreas) cuando se usan valores numéricos.

### 4. División Sintética de Polinomios
Automatización del método de Ruffini para dividir polinomios.
* **Capacidad:** Soporta coeficientes y raíces tanto reales como complejos.
* **Salida:** Muestra la tabla de proceso detallada, el cociente y el residuo.

### 5. Traductor de Expresiones Algebraicas
Puente entre el lenguaje natural y el matemático.
* **Función:** Convierte frases como "La suma de los cuadrados de a y b" a su ecuación correspondiente y viceversa, utilizando una base de datos de patrones.

### 6. Calculadora de Variables
Evaluador de expresiones personalizadas.
* **Uso:** Permite declarar variables ($w1=5, w2=10$) y ejecutar ecuaciones complejas que dependan de ellas.

## Tecnologías Utilizadas

* **HTML5:** Estructura semántica.
* **CSS3:** Diseño responsivo (Grid/Flexbox), animaciones CSS y variables (:root).
* **JavaScript (ES6+):** Lógica de programación y manipulación del DOM.
* **Librerías Externas:**
    * `Math.js`: Para el procesamiento simbólico, manejo de números complejos y cálculos de precisión.
    * `Google Fonts`: Para las tipografías *Cinzel* y *Orbitron*.
